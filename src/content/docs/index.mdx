## Qual'è l'obiettivo di AlgoMap?

L'obiettivo di **AlgoMap** è quello di fornire una piattaforma per studiare
ed apprendere le principali strutture dati ed algoritmi. Il sito è diviso in due grosse macro aree rispettivamente per
le **strutture dati** ed **algoritmi**.

Per raggiungere il compimento di questo ambizioso obiettivo verrà data priorità alla parte di struttura dati, e poi quella di
algoritmi.
